New Features

  • Added support for Xcode 6.1
  • Added support for case insensitive file systems based on OSXFUSE.framework. By default all FUSE volumes are treated as case sensitive. See LoopbackFS commit 089a847 for details.

    Please note that disabling support for case sensitive names does not make a FUSE volume case insensitive. It only tells the system that the volume should be treated as such.

Fixes

  • Fixed an incompatibility with Comodo Antivirus. In rare cases Comodo Antivirus might trigger a kernel panic when a FUSE daemon becomes unresponsive and the volume is being unmounted.
