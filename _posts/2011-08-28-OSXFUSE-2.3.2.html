Fixes

  • Fixed a permission issue regarding Finder in macOS 10.7. Creating files on a OSXFUSE volume, that had not been mounted with option allow_other, failed due to a change in Finder. Tracked as issue #5: "Finder permission problem". Fix suggested by Peter Stegemann
  • Fixed a bug in the OSXFUSE kernel extension, that could render a file system unresponsive and therefore being ejected after a timeout. This bug affected solely Macs running a 64 bit kernel and was triggered through many simultaneous file system operations. Tracked as issue #7: "Deadlock in filesystem". Fix based on Anatol Pomozov's findings.

Changes

  • Applied Tuxera's MacFUSE 2.2.0/1 patches.
  • Introduced "fat" distribution package containing different installation packages for macOS 10.6 and 10.7.
